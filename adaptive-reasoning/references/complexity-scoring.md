# Complexity Scoring Guide

Detailed guide for assessing task complexity across multiple dimensions.

## Scoring Dimensions

### Multi-step Logic (+3)
Tasks requiring planning, proofs, or debugging chains across multiple components.

### Ambiguity (+2)
Nuanced questions with trade-offs where "it depends" is a valid answer.

### Code Architecture (+2)
System design, refactoring, or security review of interconnected components.

### Math/Formal Reasoning (+2)
Calculations, algorithms, logic puzzles, or formal verification.

### Novel Problem (+1)
No clear pattern exists; requires creative or exploratory thinking.

### High Stakes (+1)
Production changes, irreversible actions, or security-critical decisions.

## Subtractions

- Routine/repetitive task: -2
- Clear single answer: -2
- Simple lookup/fetch: -3
