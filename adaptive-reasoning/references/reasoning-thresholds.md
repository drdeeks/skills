# Reasoning Thresholds Reference

Decision thresholds for adaptive reasoning activation.

## Threshold Levels

| Score | Response Mode | Icon | Description |
|-------|--------------|------|-------------|
| 0-2 | Fast mode | None | Simple lookup, no reasoning needed |
| 3-5 | Standard | None | Light internal deliberation |
| 6-7 | Thinking | 🧠 | Extended thinking enabled |
| 8+ | Deep thinking | 🧠🔥 | Full reasoning activation |

## Activation Protocol

1. Score the request on all dimensions
2. Sum weighted scores
3. Apply subtraction modifiers
4. Determine threshold level
5. Activate appropriate reasoning mode
6. Append indicator icon to response end
