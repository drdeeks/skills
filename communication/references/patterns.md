# Communication Patterns

Common communication patterns for agent systems.

## Structured Responses

Use consistent formats across all communications:
- Headers for sections
- Tables for comparisons
- Lists for action items
- Code blocks for commands

## Escalation Patterns

1. Information only - no action needed
2. Recommendation - suggest action
3. Decision required - present options
4. Urgent - immediate attention needed

## Feedback Loops

- Acknowledge receipt
- Confirm understanding
- Report progress
- Deliver results
