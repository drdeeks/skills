# Media Subskills Reference

## gif-search
Search and discover GIFs from various sources.
- Giphy API integration
- Tenor API integration
- Local GIF management
- Category/tag filtering

## heartmula
Audio processing and music creation tools.
- Audio file manipulation
- Beat detection
- Music generation
- Format conversion

## songsee
Music discovery and playlist management.
- Spotify API integration
- Last.fm integration
- Playlist creation
- Recommendation engine

## youtube-content
YouTube content creation, transcript extraction, and video processing.
- Transcript fetching
- Content transformation (chapters, summaries, threads, blog posts)
- Video metadata extraction
- Format conversion

## Usage
Load specific sub-skill based on task:
```bash
# GIF work
media/gif-search

# Audio work
media/heartmula

# Music discovery
media/songsee

# YouTube content
media/youtube-content
```
