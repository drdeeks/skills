# Transcript Transformation Guide

## Output Formats

### Chapters
- Group transcript by topic shifts
- Include timestamps
- 5-15 word chapter titles
- Chronological order

### Summary
- 5-10 sentences
- Cover main points
- No timestamps needed
- Concise overview

### Chapter Summaries
- Chapters with paragraph each
- 3-5 sentences per chapter
- Include key details

### Twitter/X Thread
- Numbered posts (1/n)
- Each under 280 chars
- Include timestamps
- Hook in first tweet

### Blog Post
- Title + sections
- Introduction, body, conclusion
- Key takeaways section
- 800-2000 words

### Quotes
- Notable statements
- With timestamps
- Speaker attribution
- 5-10 best quotes

## Chunking Strategy
- If > 50K chars: split into ~40K chunks
- 2K overlap between chunks
- Summarize each chunk
- Merge summaries

## Language Handling
- Try requested language first
- Fallback to any available
- Note actual language in output

## Quality Checks
- [ ] Coherent flow
- [ ] Correct timestamps
- [ ] Complete coverage
- [ ] No hallucinated content
