# Diagramming Tools Overview

## Supported Formats

| Format | Extension | Use Case |
|--------|-----------|----------|
| Mermaid | .md | Text-based diagrams in markdown |
| PlantUML | .puml | UML diagrams |
| Graphviz | .dot | Graph diagrams |
| Excalidraw | .excalidraw | Hand-drawn style |

## Diagram Types

| Type | Tool | Best For |
|------|------|----------|
| Flowchart | Mermaid | Process flows |
| Sequence | Mermaid | API interactions |
| Class | PlantUML | OOP design |
| ER | Graphviz | Database schema |
| Architecture | Excalidraw | System design |

## Integration

- VS Code: Mermaid preview extension
- GitHub: Native Mermaid rendering
- Notion: Embedded diagrams
- Confluence: PlantUML plugin
