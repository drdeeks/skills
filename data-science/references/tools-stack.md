# Data Science Tools Stack

## Core Libraries

| Library | Purpose |
|---------|---------|
| pandas | Data manipulation and analysis |
| numpy | Numerical computing |
| matplotlib | Data visualization |
| scikit-learn | Machine learning |
| seaborn | Statistical visualization |

## Development Environment

| Tool | Purpose |
|------|---------|
| Jupyter Notebook | Interactive computing |
| JupyterLab | Next-gen notebook interface |
| VS Code | Code editing with Python extension |
| Google Colab | Cloud-based notebooks |

## Data Formats

| Format | Extension | Use Case |
|--------|-----------|----------|
| CSV | .csv | Tabular data |
| JSON | .json | Semi-structured data |
| Parquet | .parquet | Columnar storage |
| HDF5 | .h5 | Large numerical datasets |

## Workflow

1. **Data Collection** — Load from files, APIs, databases
2. **Data Cleaning** — Handle missing values, outliers
3. **Exploration** — Statistical summaries, visualizations
4. **Modeling** — Train/test split, model selection
5. **Evaluation** — Metrics, cross-validation
6. **Deployment** — Export model, API serving
