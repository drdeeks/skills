# Autonomous AI Agents Overview

Overview of skills for spawning and orchestrating autonomous AI coding agents.

## Agent Types

- Single agent tasks with background execution
- Multi-agent parallel workflows
- Subagent delegation and coordination

## Workflow Patterns

1. Task decomposition into subtasks
2. Agent assignment based on capability
3. Parallel execution with synchronization
4. Result aggregation and reporting

## Provider Support

All agents work with any LLM provider that supports tool use.
