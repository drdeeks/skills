# Session Management

## Lifecycle
1. **Create**: `opencode session new`
2. **Run**: Agent executes tasks
3. **Pause**: `opencode session pause`
4. **Resume**: `opencode session resume`
5. **End**: `opencode session end`

## State Persistence
- Working directory
- Environment variables
- Tool configurations
- Conversation history

## Multi-Agent Sessions
- Shared context via workspace
- Inter-agent communication
- Coordinated task execution
