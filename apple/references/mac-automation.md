# Apple macOS Automation Overview

Apple-specific automation capabilities for macOS systems.

## Supported Services

- iMessage: Send and receive messages
- Reminders: Create and manage reminders
- Notes: Access and organize notes
- FindMy: Locate devices and people
- Shortcuts: Automate system workflows

## Platform Requirements

- macOS 12.0 or later
- AppleScript or Shortcuts access
- iCloud account for cross-device sync
