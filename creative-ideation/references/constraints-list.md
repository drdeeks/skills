# Creative Constraints Library

## Technical Constraints

| Constraint | Example Prompt |
|------------|----------------|
| No loops | Write a recursive solution without any loop constructs |
| Single file | Build an entire app in one file |
| No dependencies | Create a web server using only stdlib |
| Time limit | Complete the feature in 10 minutes |
| Output only | Write a program that only prints, never reads |

## Conceptual Constraints

| Constraint | Example Prompt |
|------------|----------------|
| Reverse | Build a TODO app that removes items |
| Combine | Merge two unrelated tools into one |
| Simplify | Reduce a complex system to 3 components |
| Analogize | Describe software architecture as a restaurant |

## Anti-Patterns to Avoid

- Over-engineering simple problems
- Adding features without clear requirements
- Ignoring existing solutions
- Reinventing the wheel unnecessarily
