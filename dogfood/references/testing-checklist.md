# Dogfooding Testing Checklist

## Pre-Test Setup

- [ ] Install skill in development environment
- [ ] Review SKILL.md for completeness
- [ ] Check scripts for syntax errors
- [ ] Verify references are accessible

## Functional Testing

- [ ] Core functionality works as documented
- [ ] All scripts execute without errors
- [ ] Edge cases handled gracefully
- [ ] Error messages are helpful

## Usability Testing

- [ ] Documentation is clear
- [ ] Examples are runnable
- [ ] Installation steps work
- [ ] No hardcoded paths

## Performance Testing

- [ ] Scripts complete in reasonable time
- [ ] Memory usage is acceptable
- [ ] No infinite loops

## Reporting Issues

1. Describe the problem clearly
2. Include steps to reproduce
3. Show expected vs actual behavior
4. Include error messages
5. Suggest fixes if possible
