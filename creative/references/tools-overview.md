# Creative Tools Overview

## Available Subskills

| Skill | Purpose |
|-------|---------|
| ascii-art | Generate ASCII art from text or images |
| ascii-video | Create animated ASCII art from video |
| creative-ideation | Brainstorming and ideation techniques |
| excalidraw | Create diagrams and sketches |
| songwriting-and-ai-music | Generate music and lyrics |

## Tools & Libraries

| Tool | Type | License |
|------|------|---------|
| ASCII art generators | CLI/Python | MIT |
| Excalidraw | Web-based | MIT |
| Tone.js | Audio synthesis | MIT |
| MuseScore | Music notation | GPL |

## Usage Patterns

1. **Quick ASCII Art**: Use ASCII art generators for text-based visuals
2. **Diagrams**: Use Excalidraw for hand-drawn style diagrams
3. **Music**: Use Tone.js for browser-based music creation
4. **Lyrics**: Use AI models for lyrical suggestions
