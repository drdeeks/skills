# Blockchain

Blockchain development skills for Base and Solana networks.

### Base (Ethereum L2)
- Smart contract deployment
- Wallet management via MPC
- On-chain transactions
- Token operations

### Solana
- Program deployment
- Account management
- Token operations
- RPC interactions
