# Blockchain Development Overview

Overview of blockchain development skills for Base and Solana networks.

## Base (Ethereum L2)

- Smart contract deployment via Foundry/Hardhat
- Wallet management via MPC providers
- On-chain transactions and token operations
- ERC-20/ERC-721/ERC-1155 standards

## Solana

- Program deployment via Anchor/Solana CLI
- Account management and PDAs
- Token operations (SPL Token)
- RPC interactions

## Common Tools

- Foundry for Ethereum development
- Anchor for Solana development
- Hardhat for multi-chain support
