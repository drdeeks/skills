# Base Blockchain Commands Reference

Quick reference for Base blockchain queries.

## Commands

| Command | Description |
|---------|-------------|
| `wallet <address>` | Portfolio with USD pricing |
| `tx <hash>` | Transaction details |
| `token <contract>` | ERC-20 token info |
| `gas` | Gas analysis |
| `contract <address>` | Contract inspection |
| `whales` | Large transfer detection |
| `stats` | Network health |
| `price <symbol>` | Price lookup |

## Known Token Symbols

ETH, WETH, USDC, cbETH, AERO, DEGEN, TOSHI, BRETT, WELL, wstETH, rETH, cbBTC
