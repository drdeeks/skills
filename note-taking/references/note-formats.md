# Note Taking - Supported Formats

## Markdown (.md)
Standard markdown with frontmatter support:
```markdown
---
title: Note Title
tags: [tag1, tag2]
date: 2026-01-15
---

# Note Title

Content here...
```

## Plain Text (.txt)
Simple text files for quick notes.

## JSON (.json)
Structured data notes:
```json
{
  "title": "Note Title",
  "content": "Note content",
  "tags": ["tag1", "tag2"],
  "metadata": {}
}
```

## YAML (.yaml/.yml)
Configuration-style notes:
```yaml
title: Note Title
content: |
  Multi-line
  content
tags: [tag1, tag2]
```
