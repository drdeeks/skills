# inference.sh API Catalog

## Image Generation
- **FLUX**: High-quality image generation
- **Reve**: Photorealistic images
- **Seedream**: Creative/artistic images
- **Grok Imagine**: X's image model
- **Gemini**: Google's image generation

## Video Generation
- **Veo**: Google's video generation
- **Wan**: High-quality video
- **Seedance**: Dance/motion video
- **OmniHuman**: Human-centric video
- **HunyuanVideo**: Tencent's video model

## LLMs
- **Claude**: Anthropic models (via OpenRouter)
- **Gemini**: Google models
- **Kimi**: Moonshot AI
- **GLM-4**: Zhipu AI

## Search
- **Tavily**: AI-optimized search
- **Exa**: Neural search

## 3D
- **Rodin**: 3D model generation

## Social
- **Twitter/X**: Automation tools

## Audio
- **TTS**: Text-to-speech
- **Voice Cloning**: Voice synthesis
