# Obsidian - Templates

## Daily Note Template
```markdown
---
date: {{date}}
tags: [daily]
---

# {{date}}

## Morning
- [ ] Task 1
- [ ] Task 2

## Notes

## Evening Review
- What went well?
- What could be improved?
```

## Project Note Template
```markdown
---
project: Project Name
status: active
tags: [project]
---

# Project Name

## Overview

## Goals
- [ ] Goal 1
- [ ] Goal 2

## Tasks
- [ ] Task 1
- [ ] Task 2

## Resources

## Meetings
```
