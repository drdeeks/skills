# Debugging Tools

## Language-Specific
- **Python**: pdb, ipdb, pytest --pdb
- **JavaScript**: Chrome DevTools, node --inspect
- **Go**: delve, pprof
- **Rust**: gdb, lldb, rust-analyzer

## General
- **Logging**: Structured logs (JSON)
- **Metrics**: Prometheus, Datadog
- **Tracing**: OpenTelemetry, Jaeger
- **Profiling**: py-spy, perf, flamegraphs

## Remote Debugging
- SSH port forwarding
- VS Code Remote
- Kubernetes port-forward
- Cloud provider tools

## Post-Mortem
- Core dumps
- Memory analysis
- Thread dumps
- Network captures
