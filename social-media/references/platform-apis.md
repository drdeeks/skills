# Social Media Platform APIs

## X (Twitter) API
- **v2**: Current API with OAuth 2.0
- **Endpoints**: Tweets, Users, Spaces, Lists
- **Rate Limits**: 300 req/15min (standard)
- **Auth**: Bearer token / OAuth 2.0

## Mastodon API
- **Standard**: ActivityPub compatible
- **Endpoints**: Toots, Accounts, Timelines
- **Rate Limits**: 300 req/5min (default)
- **Auth**: OAuth 2.0

## LinkedIn API
- **Marketing API**: Ads, content posting
- **Profile API**: Read-only profile data
- **Auth**: OAuth 2.0
- **Rate Limits**: Varies by endpoint

## Bluesky (AT Protocol)
- **Lexicon**: Schema-based API
- **Endpoints**: Posts, Feeds, Moderation
- **Auth**: OAuth 2.0 / App passwords
- **Decentralized**: Federation support
