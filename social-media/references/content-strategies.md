# Social Media Content Strategies

## Posting Frequency
- **X/Twitter**: 3-5 posts/day
- **LinkedIn**: 1 post/day
- **Mastodon**: 2-3 posts/day
- **Bluesky**: 2-4 posts/day

## Content Types
- **Educational**: Tips, tutorials, insights
- **Engagement**: Questions, polls, discussions
- **Promotional**: Product updates, announcements
- **Curated**: Industry news, resources

## Best Practices
- Use relevant hashtags (3-5 max)
- Include media (images/video) when possible
- Engage with replies promptly
- Schedule for peak audience hours
- Monitor analytics weekly

## Automation
- Buffer/Hootsuite for scheduling
- Zapier/Make for cross-posting
- Custom scripts for analytics
