# Skill Lifecycle

## Creation
1. Run `skill-creator` to scaffold
2. Add scripts and references
3. Write documentation
4. Validate with `skill-creator-pro`

## Maintenance
- Update references when APIs change
- Fix validation issues promptly
- Keep dependencies current
- Monitor for deprecations

## Deprecation
- Mark as deprecated in SKILL.md
- Provide migration path
- Remove after 6 months
