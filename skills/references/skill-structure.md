# Skill Structure

## Required Directories
- `scripts/` - 2+ executable scripts (.py, .sh, .js, .ts)
- `references/` - 3+ reference files (.md, .txt, .json, .yaml, .yml, .csv)

## Required Files
- `SKILL.md` - Main skill documentation with YAML frontmatter

## Frontmatter Requirements
```yaml
---
name: skill-name
description: "Skill description"
license: MIT
---
```

## Optional
- `templates/` - Template files
- Subdirectories for sub-skills
