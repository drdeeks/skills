# Ethereum Development Tools

## Development Frameworks

| Tool | Purpose |
|------|---------|
| Foundry | Fast, portable, modular toolkit for Ethereum development |
| Hardhat | Development environment for compiling, testing, and deploying |
| Remix | Browser-based IDE for smart contract development |

## Testing Tools

| Tool | Purpose |
|------|---------|
| Forge Test | Run Solidity tests |
| Hardhat Test | JavaScript/TypeScript testing |
| Tenderly | Transaction simulation and debugging |

## Deployment Tools

| Tool | Purpose |
|------|---------|
| Foundry Deploy | Script-based deployment |
| Hardhat Ignition | Declarative deployment system |
| OpenZeppelin Defender | Secure deployment and management |
