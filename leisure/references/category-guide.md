# Leisure Skills Category Guide

## Purpose
This category contains skills related to leisure activities, entertainment, and lifestyle.

## Subskills

### find-nearby
Location-based discovery for nearby points of interest, events, and activities.

## Adding New Leisure Skills
1. Create subdirectory under leisure/
2. Add SKILL.md with proper frontmatter
3. Include scripts/ and references/ directories
4. Update this category guide

## Categories
- **Food & Drink**: Restaurants, cafes, recipes
- **Entertainment**: Movies, games, events
- **Outdoor**: Parks, hiking, activities
- **Social**: Meetups, communities
- **Wellness**: Spa, fitness, relaxation
