# Domain Registration Guide

| Registrar | Features | Price Range |
|-----------|----------|-------------|
| Cloudflare | Free WHOIS, DNS | At-cost pricing |
| Namecheap | Wide selection | $8-15/year |
| Google Domains | Simple interface | $12-20/year |
| Porkbun | Competitive pricing | $5-12/year |

## Registration Process

1. **Choose domain name**
   - Keep it short and memorable
   - Avoid trademark conflicts
   - Consider .com, .io, .dev extensions

2. **Check availability**
   - Use WHOIS lookup
   - Check multiple registrars

3. **Configure DNS**
   - Set nameservers
   - Add DNS records

4. **Enable privacy protection**
   - Use WHOIS privacy
   - Enable auto-renew

## Best Practices

1. Enable auto-renewal
2. Use WHOIS privacy
3. Keep contact info updated
4. Set calendar reminders for renewal
5. Use strong account passwords
