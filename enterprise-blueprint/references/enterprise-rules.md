# Enterprise Rules

## Required Parts
- Part I: Overview
- Part II: Architecture
- Part III: Implementation
- Part IV: Testing
- Part V: Deployment

## Rollback Tags
Every section must have rollback tags for safe deployment.

## Change Log
All changes must be documented in a change log.

## Module Registry
All modules must be registered with version and dependencies.

## Feature Flags
Feature flags must be documented and tracked.

## Performance Budgets
Performance budgets must be defined and enforced.

## Testing Requirements
- Unit tests for all modules
- Integration tests for critical paths
- End-to-end tests for user flows

## Error Catalog
Common errors and recovery procedures:
- Missing required section
- Invalid rollback tag format
- Missing change log entry
- Incomplete module registry
